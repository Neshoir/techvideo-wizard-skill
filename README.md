# TechVideo Wizard - SuperMap GIS技术小视频脚本向导

通过 6 步结构化引导，快速生成专业产品视频旁白脚本。

## 功能特点

- **渐进式引导**：每步只问一个问题，避免信息过载
- **结构化输出**：自动生成「开头 → 方案 → 效果 → 结果 → 收尾」的完整脚本
- **量化约束**：强制要求具体、可量化的表达，拒绝空话
- **术语约束**：支持固定术语搭配，确保品牌一致性
- **一键优化**：支持对生成内容进行自动润色

## 适用场景

- 产品演示视频脚本
- 技术功能介绍视频
- 演示系统操作流程
- 营销宣传短片

## 触发方式

说出以下任意表达即可激活：

```
帮我写个 xxx 小视频的脚本
帮我写个产品介绍视频的脚本
视频脚本向导
```

## 使用流程

| 步骤 | 内容 | 类型 |
|------|------|------|
| Step 1 | 一句话介绍 | 必填 |
| Step 2 | 历史 vs 当前对比 | 可选 |
| Step 3 | 核心步骤 | 必填 |
| Step 4 | 变化/效果 | 可选 |
| Step 5 | 最终效果 | 必填 |
| Step 6 | 生成脚本 | 自动 |
| Step 7 | 自动优化 | 可选 |

## 术语约束

脚本生成时会自动保留固定术语搭配，确保品牌一致性：
- 产品名称：SuperMap GIS、SuperMap iDesktop 等
- 技术术语：三维场景、倾斜摄影、BIM模型 等
- 功能模块：数据导入、服务发布 等

详细术语表见 `TERM_CONSTRAINTS.md`
